# 네온 시티 시뮬레이터

## 실행
```bash
npm install
npm run dev
```

## 플레이 방법
- 좌측 패널에서 타일을 선택하고 중앙 40x40 맵에 배치합니다.
- 상단에서 시간 배속(일시정지/1x/3x/10x)을 조절합니다.
- 우측 패널에서 경제/전력/교통/오염/행복 지표와 차트를 확인합니다.
- 랜덤 이벤트가 발생하면 선택지로 도시 정책을 결정합니다.
- 저장/불러오기/JSON 내보내기/가져오기/초기화를 통해 도시를 관리합니다.

## 시뮬레이션 규칙
- 주거는 수용량, 상업/산업은 일자리, 발전소/태양광은 전력 생산을 담당합니다.
- 도로 인접 건물은 활성화되어 더 큰 영향을 냅니다.
- 오염·교통·범죄가 높으면 행복/건강이 하락합니다.
- 병원/학교/경찰서는 건강/교육/치안을 개선합니다.
- 전력 부족 시 세수와 성장률이 감소합니다.
- 행복도, 일자리 비율, 주거 여유가 인구 성장률을 결정합니다.
